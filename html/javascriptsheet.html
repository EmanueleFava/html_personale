JavaScript Cheat Sheet

Operatori di Comparazione

  • ==: verifica l'uguaglianza "debole", che consente il "coercion" dei tipi.
    Esempio: 5 == '5' è vero.
  • ===: verifica l'uguaglianza "rigorosa", dove i tipi devono essere uguali oltre che i valori.
    Esempio: 5 === '5' è falso.
  • !=: verifica la disuguaglianza "debole", simile a == ma al contrario.
    Esempio: 5 != '5' è falso.
  • !==: verifica la disuguaglianza "rigorosa", simile a === ma al contrario.
    Esempio: 5 !== '5' è vero.
  • Altri operatori di comparazione includono < (minore di), <= (minore o uguale a), > (maggiore di), >= (maggiore o uguale a).
console.log(10 == '10'); // vero, coerzione del tipo
console.log(10 === '10'); // falso, tipi diversi
console.log(10 != '10'); // falso, coerzione del tipo
console.log(10 !== '10'); // vero, tipi diversi
console.log(5 < 10); // vero
console.log(10 >= 10); // vero

Variabili e Tipi di Dati

  • let: dichiara una variabile che può essere riassegnata.
  • const: dichiara una costante (non modificabile).
  • var: dichiarazione di variabile (vecchio stile, non raccomandato).
  • String: testo (racchiuso tra virgolette).
  • Number: numeri interi o decimali.
  • Boolean: valori vero o falso.
let nome = 'Mario';
const eta = 30;
var indirizzo = 'Via Roma';
let isOnline = true;
let prezzo = 9.99;

Funzioni

  • function: dichiara una funzione.
  • arrow function: sintassi breve per le funzioni.
  • return: restituisce un valore.
  • parametri: variabili passate alla funzione.
function saluta(nome) {
    return 'Ciao ' + nome;
}

const add = (a, b) => a + b;

Condizioni

  • if: esegue il blocco se la condizione è vera.
  • else: esegue il blocco se la condizione è falsa.
  • else if: permette condizioni multiple.
let ora è 10;
if (ora < 12) {
    console.log('Buongiorno');
} else if (ora < 18) {
    console.log('Buon pomeriggio');
} else {
    console.log('Buonasera');
}

Cicli

  • for: esegue un ciclo per un numero definito di volte.
  • while: esegue il ciclo finché la condizione è vera.
  • for...of: esegue un ciclo su ogni elemento di un array.
  • for...in: esegue un ciclo su ogni chiave di un oggetto.
for (let i = 0; i < 5; i++) {
    console.log(i);
}

let count = 0;
while (count < 3) {
    console.log(count);
    count++;
}

Array e Oggetti

  • Array: una lista di elementi.
  • push: aggiunge un elemento alla fine dell'array.
  • pop: rimuove l'ultimo elemento dell'array.
  • Oggetto: una collezione di coppie chiave-valore.
let frutta = ['mela', 'banana', 'arancia'];
frutta.push('pera');
frutta.pop();

let persona = { nome: 'Mario', eta: 30 };
persona.indirizzo è 'Via Roma';

Spread Operator

  • Spread Operator (...): consente di espandere elementi, creare copie di array o oggetti.
  • Creare una copia di un array: utilizzando `...` per copiare un array.
  • Combinare array: è possibile unire più array con lo spread operator.
let arrayOriginale = [1, 2, 3];
let arrayCopia = [...arrayOriginale]; // Copia di array

let arrayUno = [4, 5];
let arrayCombinato = [...arrayOriginale, ...arrayUno]; // Combinare array

Manipolazione del DOM

  • document.getElementById: ottiene un elemento per ID.
  • document.querySelector: ottiene il primo elemento che corrisponde a un selettore CSS.
  • addEventListener: aggiunge un listener per gli eventi.
let elemento è document.getElementById('myId');
let btn è document.querySelector('.myButton');
btn.addEventListener('click', function() {
    console.log('Bottone cliccato');
});

Console

  • console.log: stampa messaggi o variabili.
  • console.error: stampa errori.
  • console.warn: stampa avvisi.
console.log('Messaggio di log');
console.error('Errore');
console.warn('Avviso');
