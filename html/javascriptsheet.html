JavaScript Cheat Sheet

Operatori di Comparazione

  • ==: verifica l'uguaglianza "debole", consentendo la coercizione dei tipi.
    Esempio: 5 == '5' è vero.
  • ===: verifica l'uguaglianza "rigorosa", dove i tipi devono essere uguali oltre ai valori.
    Esempio: 5 === '5' è falso.
  • !=: verifica la disuguaglianza "debole", simile a == ma al contrario.
    Esempio: 5 != '5' è falso.
  • !==: verifica la disuguaglianza "rigorosa", simile a === ma al contrario.
    Esempio: 5 !== '5' è vero.

Variabili e Tipi di Dati

  • let: dichiara una variabile che può essere riassegnata.
  • const: dichiara una costante (non modificabile).
  • var: dichiarazione di variabile (vecchio stile, non raccomandato).
  • String: testo (racchiuso tra virgolette).
  • Number: numeri interi o decimali.
  • Boolean: valori vero o falso.
  • null: indica un valore assente o vuoto.
  • undefined: indica una variabile dichiarata ma senza valore assegnato.
let nome = 'Mario';
const eta = 30;
var indirizzo = 'Via Roma';
let isOnline = true;
let prezzo = 9.99;
let nulla = null;
let senzaDefinizione; // undefined

Funzioni

  • function: dichiara una funzione.
  • arrow function: sintassi breve per le funzioni.
  • high-order functions: funzioni che prendono altre funzioni come parametri o restituiscono funzioni.
  • callback: una funzione passata come argomento a un'altra funzione.
function saluta(nome) {
    return 'Ciao ' + nome;
}

const add = (a, b) => a + b;

function eseguiCallback(callback) {
    callback('Funzione di callback');
}
eseguiCallback(console.log); // Funzione di callback

Manipolazione del DOM

  • document.getElementById: ottiene un elemento per ID.
  • document.querySelector: ottiene il primo elemento che corrisponde a un selettore CSS.
  • document.querySelectorAll: ottiene tutti gli elementi che corrispondono a un selettore CSS.
  • addEventListener: aggiunge un listener per gli eventi.
  • removeEventListener: rimuove un listener per gli eventi.
  • setAttribute: imposta un attributo su un elemento.
  • getAttribute: ottiene un attributo di un elemento.
  • removeChild: rimuove un elemento dal DOM.
  • cloneNode: crea una copia di un nodo. Passando `true` copia anche i nodi figli; `false` copia solo il nodo stesso.
let elemento = document.querySelector('.myClass');
elemento.setAttribute('title', 'Nuovo titolo');
console.log(elemento.getAttribute('title')); // "Nuovo titolo"

let elementoPadre = document.getElementById('parent');
let elementoFiglio = document.querySelector('.child');
elementoPadre.removeChild(elementoFiglio);

let nodoOriginale = document.getElementById('original');
let nodoCopia = nodoOriginale.cloneNode(false); // Copia solo il nodo stesso senza eventi

Console

  • console.log: stampa messaggi o variabili.
  • console.error: stampa errori.
  • console.warn: stampa avvisi.
console.log('Messaggio di log');
console.error('Errore');
console.warn('Avviso');
